#+TITLE: DOOM EMACS — ŚCIĄGA UŻYTKOWA (MINIMAL)
#+STARTUP: overview
#+OPTIONS: toc:nil

* Zasada ogólna
- SPC = klawisz główny (leader)
- ESC = wyjście z trybu wstawiania
- Gdy nie wiesz co zrobić → SPC h d h

 #+begin_quote
EN: General rules. SPACE is the leader key, ESC exits insert mode.
If unsure, open Doom help with SPC h d h.
#+end_quote

* Pliki i bufory
** Pliki
- Otwórz plik :: SPC f f
- Ostatnie pliki :: SPC f r
- Zapisz plik :: SPC f s

 #+begin_quote
EN: File operations. Open a file, reopen recent files, and save the current buffer using leader key shortcuts.
#+end_quot

** Bufory
- Przełącz bufor :: SPC b b
- Zamknij bufor :: SPC b d
- Lista buforów :: SPC b i

 #+begin_quote
EN: Buffer management. Switch between buffers, close the current buffer, or open the buffer list using leader key shortcuts.
#+end_quote

* Dired (pliki i katalogi)
** Otwieranie
- Otwórz dired w katalogu pliku :: SPC f d
- Otwórz dired ręcznie :: M-x dired

 #+begin_quote
EN: Dired (directory editor). Open Dired in the current file’s directory or start it manually via M-x.
#+end_quote

** Nawigacja
- Wejdź do katalogu :: RET
- Cofnij katalog :: ^
- Otwórz w tym samym oknie :: RET
- Otwórz w nowym oknie :: o

 #+begin_quote
EN: Navigation in Dired. Enter directories, go up to the parent directory,
open files in the same window, or open them in a new window.
#+end_quote

** Operacje na plikach
- Zaznacz plik :: m
- Odznacz plik :: u
- Odznacz wszystko :: U

- Kopiuj :: C
- Przenieś / zmień nazwę :: R
- Usuń :: D
- Utwórz katalog :: +

 #+begin_quote
EN: File operations in Dired. Mark and unmark files, then copy, move/rename,
delete them, or create new directories.
#+end_quote

** Widok
- Odśwież :: g
- Pokaż / ukryj szczegóły :: (

 #+begin_quote
EN: View options in Dired. Refresh the buffer and toggle file details.
#+end_quote

* Okna (pane)

- Podziel pionowo :: SPC w v
- Podziel poziomo :: SPC w s
- Zamknij okno :: SPC w d
- Przełącz okna :: SPC w w
- Zostaw jedno okno :: SPC w m
- Przełączanie między oknami :: SPC w h/l
- Zamiana okien miejscami :: SPC w r

 #+begin_quote
EN: Window management. Split windows vertically or horizontally, switch between
windows, close them, keep only one window, or rotate window contents.
#+end_quote

* Edycja tekstu (evil)
** Tryby
- Wstawianie :: i
- Wyjście :: ESC

 #+begin_quote
EN: Text editing with Evil. Enter insert mode and exit back to normal mode.
#+end_quote

** Operacje
- Usuń linię :: dd
- Kopiuj :: y
- Wklej :: p
- Cofnij :: u
- Przywróć :: C-r

 #+begin_quote
EN: Basic editing operations in Evil. Delete lines, yank (copy), paste,
undo changes, and redo them.
#+end_quote

** Zaznaczanie
- Linia :: V
- Blok :: C-v

 #+begin_quote
EN: Visual selection modes in Evil. Select entire lines or rectangular blocks.
#+end_quote

* Szukanie
- Szukaj w pliku :: /
- Następne trafienie :: n
- Poprzednie :: N
- Szukaj w projekcie :: SPC /

 #+begin_quote
EN: Search operations. Search within the current buffer, jump to next or
previous matches, or search across the project.
#+end_quote

* Projekty
- Przełącz projekt :: SPC p p
- Plik w projekcie :: SPC p f
- Szukaj w projekcie :: SPC p s

 #+begin_quote
EN: Project management. Switch between projects, open files within a project,
or search across the project.
#+end_quote

* Org-mode (minimum)

** Struktura
- Nowy nagłówek :: *
- Zwiń / rozwiń :: TAB

 #+begin_quote
EN: Basic Org-mode structure. Create headings and fold or unfold sections.
#+end_quote

** Zadania
- Zmień TODO :: SPC m t
- Oznacz DONE :: SPC m t d

#+begin_quote
EN: Task management in Org-mode. Change the TODO state and mark tasks as DONE.
#+end_quote

** Agenda
- Agenda :: SPC o A

#+begin_quote
EN: Org Agenda. Open the agenda view to see tasks and scheduled items.
#+end_quote

* Org-journal
- Dzisiejszy wpis :: M-x org-journal-new-entry
- Poprzedni dzień :: M-x org-journal-open-previous-entry

Zasada:
- jeden dzień = jeden plik
- zero struktury, tylko zapis

#+begin_quote
EN: Daily journal entries. Create a new entry for today or move to the previous day.
One day equals one file. No structure — just write.
#+end_quote

* Org-roam (minimum konieczne)
- Znajdź / utwórz notatkę :: SPC n r f
- Wstaw link :: SPC n r i

Zasada:
- nie ucz się więcej skrótów
- reszta to opcjonalny hałas

#+begin_quote
EN: Org-roam basics. Find or create notes and insert links between them.
Do not learn more shortcuts — everything else is optional noise.
#+end_quote

* Terminal
- Otwórz vterm :: SPC o t
- Zamknij terminal :: exit

#+begin_quote
EN: Terminal access. Open vterm from Doom and close the terminal by exiting the shell.
#+end_quote

* Konfiguracja Dooma
** Kiedy co robić
- Zmiany w init.el / packages.el :: doom sync + restart
- Zmiany w config.el :: SPC h r r

Zasada:
- gdy nie wiesz → doom sync

 #+begin_quote
EN: Doom configuration workflow. Change init.el or packages.el for modules and
packages, edit config.el for behavior. If unsure, run doom sync.
#+end_quote

* Pomoc / ratunek
- Opis skrótów :: SPC h d h
- Dokumentacja modułu :: kursor na nazwie + K
- Reload konfiguracji :: SPC h r r

 #+begin_quote
EN: Help and recovery. Browse keybindings, open module documentation at point,
or reload Doom configuration when things break.
#+end_quote

* Złota zasada
Jeśli coś Ci pomaga — jest poprawne.
Jeśli przeszkadza — wyłącz.

#+begin_quote
EN: Golden rule. If something helps you, it is correct.
If it gets in your way, turn it off.
#+end_quote
